D4T1 [SDOI2014]旅行

传送门

Desciption:

给定一颗 nnn 个节点的树，每个节点都有一个权值和一种颜色。

有 qqq 个询问：

CC x c 将节点 xxx 的颜色改为 ccc

CW x w 将节点 xxx 的权值改为 www

QS x y 询问 <x,y><x,y><x,y> 路径上与 x,yx,yx,y 颜色相同的点的权值和

QM x y 询问 <x,y><x,y><x,y> 路径上与 x,yx,yx,y 颜色相同的点的最大权值

Solution:

首先明确肯定用树剖来维护询问和修改。

对于每一个宗教我们都要开一颗线段树来维护，所以考虑用主席树。

主席树+树剖，其他的与线段树+树剖一样。

Hint:

  1. 查询最大最小值得时候记得加上if(qr<l||ql>r) return 0;，不然不晓得飞到哪里去。
  2. 主席树的空间玄学问题。本题的 maxnmaxnmaxn 为 1e51e^51e5 ，理论上要开到 maxn<<7maxn<<7maxn<<7 的范围才保险，但是 SLOJ上 maxn<<6maxn<<6maxn<<6 才能过。
